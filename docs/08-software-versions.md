# Software versions

**Summary:** Versions of PHP, MySQL, Dojo, Font Awesome, and JS/CSS constraints used by the BGA framework, plus the list of PHP extensions available in BGA Studio.

**Source:** [Studio • Board Game Arena – Software Versions](https://boardgamearena.com/doc/Studio)

---

## Framework versions

| Technology | Version / note |
|------------|----------------|
| **Dojo Toolkit** | 1.15 |
| **PHP** | 8.4 |
| **SQL** | MySQL 5.7 |
| **JS/CSS/HTML** | Limited by minimization tools; see [JS minimization](https://boardgamearena.com/doc/Game_interface_logic:_yourgamename.js#Javascript_minimization_.28after_July_2020.29) |
| **Font Awesome** | 4.7 – [icons](https://fontawesome.com/v4.7/icons/) – use as `<i class="fa fa-clock" />` |
| **Font Awesome** | 6.4.0 – [icons (free)](https://fontawesome.com/v6/search?o=r&m=free) – use as `<i class="fa6 fa6-clock" />` |

## PHP extensions (BGA Studio)

As of May 8th, 2022, the following PHP extensions are in use in BGA Studio and available:

date, libxml, openssl, pcre, zlib, filter, hash, Reflection, SPL, session, standard, sodium, apache2handler, mysqlnd, PDO, xml, apcu, bz2, calendar, ctype, curl, dom, mbstring, FFI, fileinfo, ftp, gd, gettext, gmp, iconv, igbinary, json, exif, msgpack, mysqli, pdo_mysql, apc, posix, readline, shmop, SimpleXML, sockets, sysvmsg, sysvsem, sysvshm, tokenizer, v8js, xmlreader, xmlwriter, xsl, zip, Phar, memcached, Zend OPcache

*(Check the [Studio](https://boardgamearena.com/doc/Studio) page for any later updates.)*
