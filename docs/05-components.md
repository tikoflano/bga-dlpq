# BGA Studio game components

**Summary:** Official and unofficial JS and PHP components you can use in your game (Deck, Stock, Counter, Draggable, bga-cards, etc.). Each item links to the official doc for details. For how these fit with Dojo, TypeScript, and other frontend libraries, see [11-frontend-libraries.md](11-frontend-libraries.md).

**Source:** [Studio • Board Game Arena – BGA Studio game components reference](https://boardgamearena.com/doc/Studio)

---

## Official components

### PHP

| Component | Description | Doc |
|-----------|-------------|-----|
| **Deck** | Manage cards: deck, hands, pick, move, shuffle | [Deck](https://boardgamearena.com/doc/Deck) |
| **PlayerCounter and TableCounter** | Manage counters (e.g. score, resources) | [PlayerCounter and TableCounter](https://boardgamearena.com/doc/PlayerCounter_and_TableCounter) |

### JS

| Component | Description | Doc |
|-----------|-------------|-----|
| **Counter** | Counter that can increase/decrease (e.g. score) | [Counter](https://boardgamearena.com/doc/Counter) |
| **Draggable** | Drag'n'drop actions | [Draggable](https://boardgamearena.com/doc/Draggable) |
| **ExpandableSection** | Block of HTML that can be shown/hidden | [ExpandableSection](https://boardgamearena.com/doc/ExpandableSection) |
| **Scrollmap** | Scrollable game area (e.g. large/infinite board) | [Scrollmap](https://boardgamearena.com/doc/Scrollmap) |
| **Stock** | Set of game elements at a position | [Stock](https://boardgamearena.com/doc/Stock) |
| **Zone** | Zone where elements can enter/leave, displayed together | [Zone](https://boardgamearena.com/doc/Zone) |
| **bga-animations** | Animations | [BgaAnimations](https://boardgamearena.com/doc/BgaAnimations) |
| **bga-cards** | Cards (display, animate, stocks: deck, hand, line, grid, slot, etc.) | [BgaCards](https://boardgamearena.com/doc/BgaCards) → [12-component-bga-cards.md](12-component-bga-cards.md) |
| **bga-dice** | Dice | [BgaDice](https://boardgamearena.com/doc/BgaDice) |
| **bga-autofit** | Fit text in a fixed-size div | [BgaAutofit](https://boardgamearena.com/doc/BgaAutofit) |
| **bga-score-sheet** | Animated score sheet at end of game | [BgaScoreSheet](https://boardgamearena.com/doc/BgaScoreSheet) |
